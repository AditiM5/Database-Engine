# README

### Running test :

```make test.out```

To run the driver, type

```test.out```

and then follow the on-screen instructions.

### Running gtest :

```make gtest```

To run the driver, type

```./BigQTest```

and then follow the on-screen instructions.

### Running docker:

#### To run the main file:
```./docker.sh main```


#### To run test.out
```./docker.sh test```


#### To run gtest
```./docker.sh gtest```
